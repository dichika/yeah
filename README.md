yeah
=====

yeah connects R users with voice.
## Installation

```r
#install.packages("remotes")
remotes::install_github("dichika/yeah")
```

## Example
```r
library(yeah)
yeah("yeah")
```

## Usage
yeah(pkgname)

## Arguments

### pkgname
The package name which you want to listen to.
